# Expense Tracker

A Full Stack Expense Tracker web application.

## Features

- User Login
- User Signup
- Add Transaction
- Edit Transaction
- Delete Transaction
- Search Transaction
- Filter Income & Expense
- Dashboard Summary
- Expense Chart
- Responsive Design

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
