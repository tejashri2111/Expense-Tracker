# Expense Tracker

A full-stack Expense Tracker web application that helps users manage their daily income and expenses.

## Features

- User Login & Signup
- Add Income & Expense
- Edit Transactions
- Delete Transactions
- Search Transactions
- Filter by Income/Expense
- Dashboard with Balance, Income & Expense
- Expense Chart using Chart.js
- Responsive UI
- Form Validation

## Tech Stack

Frontend
- HTML
- CSS
- JavaScript

Backend
- Node.js
- Express.js

Database
- MongoDB
